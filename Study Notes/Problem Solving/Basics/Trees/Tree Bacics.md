# Height of Tree
Height of tree is calculated with the maximum height of left subtree or right subtree added with 1 ( the root node ), mathematical formula for it is;

The height formula is: $$
\boxed{
\text{height}(node) = 1 + \max\bigl(\text{height}(node.left),\ \text{height}(node.right)\bigr)
}
$$
